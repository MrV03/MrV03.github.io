# Vivek Maharaj Portfolio

A Next.js 14, Tailwind CSS, Three.js, and Framer Motion powered digital identity and command-center portfolio.

## Quick Start (Deploy to GitHub Pages)

1. Extract this ZIP file.
2. Push the contents to a new GitHub repository on `main` branch.
3. Go to Repository **Settings** > **Pages**.
4. Set **Source** to **GitHub Actions**.
5. Push any commit to `main` and your site will automatically build and publish to GitHub Pages!

## Local Development

```bash
npm install
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
